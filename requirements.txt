# EduConnect Haïti - Prérequis du projet
# ===========================================
# Ce projet est une application JavaScript/TypeScript (React + Vite),
# pas un projet Python. Il n'utilise donc pas pip.
# Ce fichier liste les prérequis et dépendances pour information.

# --- ENVIRONNEMENT REQUIS ---
# Node.js          >= 20.0.0
# pnpm             >= 9.0.0   (gestionnaire de paquets du monorepo)
# Git              >= 2.30

# --- INSTALLATION ---
# 1. Cloner le projet :
#      git clone https://github.com/Rocky1324/Edu_Connect.git
#      cd Edu_Connect
# 2. Installer toutes les dépendances :
#      pnpm install
# 3. Lancer l'application web en mode développement :
#      pnpm --filter @workspace/educonnect-haiti run dev
# 4. Construire pour la production :
#      pnpm --filter @workspace/educonnect-haiti run build

# --- STRUCTURE DU MONOREPO ---
# artifacts/educonnect-haiti     Application web (frontend React + Vite)
# artifacts/api-server           Serveur API (Express, optionnel)
# artifacts/mockup-sandbox       Bac à sable de prototypage UI
